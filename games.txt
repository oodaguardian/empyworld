
SYSTEM ROLE:
You are a senior full-stack developer and game developer building a free, safe,
educational kids app for a 5-year-old girl. You will install dependencies, fetch

PHASE 1 — PROJECT SETUP
════════════════════════════════════════

Initialize a React Native + Expo project (or React web app if mobile web):

  npx create-expo-app KiddieApp --template blank
  cd KiddieApp

Install all required dependencies:

  # Navigation
  npx expo install @react-navigation/native @react-navigation/stack
  npx expo install react-native-screens react-native-safe-area-context

  # WebView (for iframes + deeplinks)
  npx expo install react-native-webview

  # Audio (for read-aloud / TTS)
  npx expo install expo-speech expo-av

  # Animations
  npx expo install react-native-reanimated lottie-react-native

  # Gestures (for drag/drop games)
  npx expo install react-native-gesture-handler

  # SVG (for coloring book, shapes)
  npx expo install react-native-svg

  # Storage (save progress, scores)
  npx expo install @react-native-async-storage/async-storage

  # Voice recognition (for voice search)
  npx expo install expo-speech @react-native-voice/voice

  # Deep linking
  npx expo install expo-linking

  # Open Library API (books) - no install needed, use fetch()
  # Scratch embed - no install needed, use WebView
  # Educandy embed - no install needed, use WebView

PHASE 4 — BUILD EACH GAME
════════════════════════════════════════

Build every game listed below from scratch using React Native + open-source assets.
Each game must have:
  ✓ Colorful, large-touch UI optimized for 5-year-old fingers
  ✓ Sound effect on correct answer (expo-av, CC0 sounds from freesound.org)
  ✓ Celebration animation on win (Lottie confetti or emoji burst)
  ✓ Simple instructions shown as emoji + short text
  ✓ Back button always visible
  ✓ Score/stars saved via AsyncStorage

── COLORING BOOK ──────────────────────
Tech: react-native-svg + color picker
Logic:
  - Render SVG animal/scene outlines (use Kenney.nl SVG animals)
  - Display a palette of 12 bright colors at bottom
  - Child taps a color, then taps a region → fills with that color
  - Use SVG fill state stored in useState per region ID
  - Include 10+ scenes: cat, dog, butterfly, flower, house, fish, star, etc.
  - "Print / Save" button uses expo-media-library to save PNG

── MEMORY MATCH ────────────────────────
Tech: React Native FlatList, Animated API
Logic:
  - 4x4 grid (16 cards = 8 pairs) of emoji animals/objects
  - Cards face-down, tap to flip (CSS flip animation via Animated)
  - Match = cards stay face up + green glow
  - No match = cards flip back after 1 second
  - Track moves + time, show star rating at end
  - Difficulty: Easy (4 pairs), Medium (8), Hard (12)

── BUBBLE POP ──────────────────────────
Tech: React Native Animated + PanResponder
Logic:
  - Bubbles float upward from bottom (Animated.timing loop)
  - Each bubble has a random emoji inside (animals, fruits, letters)
  - Child taps bubble → pop animation + sound + score++
  - Missed bubbles (reach top) → lose a life (3 lives total)
  - Speed increases every 10 pops
  - Use CC0 bubble sprite from OpenGameArt or draw with SVG circle

── RACING GAME ─────────────────────────
Tech: React Native Animated, Kenney Racing Pack assets
Logic:
  - Top-down or side-scroll race track (SVG or image background)
  - Child's car at bottom, tilt left/right using two big arrow buttons
  - Avoid obstacle cars coming from top
  - Collect star pickups for bonus points
  - Speed increases over time
  - Use Kenney.nl "Racing Pack" CC0 car sprites (PNG)

── DRESS UP ────────────────────────────
Tech: React Native Image layers + drag-and-drop (gesture handler)
Logic:
  - Base character sprite (girl, boy — open-source from OpenGameArt CC0)
  - Clothing items in scrollable tray at bottom (hats, dresses, shoes, accessories)
  - Drag items onto character to equip
  - Layered Image components (zIndex) show outfit on character
  - "Save Look" button screenshots and saves to gallery
  - 20+ clothing items minimum

── ABC LEARNING GAME ───────────────────
Tech: React Native + expo-speech
Logic:
  - Show one large letter (A-Z) at a time, uppercase + lowercase
  - Animated letter bounces in
  - "Hear it" button: expo-speech says the letter name + a word ("A is for Apple 🍎")
  - Trace the letter: Show dotted letter path, child traces with finger (PanResponder)
  - Quiz mode: Show 4 letters, hear a sound, tap the right one
  - Celebrate correct answers with confetti Lottie

── NUMBER LEARNING GAME (123s) ─────────
Tech: React Native + expo-speech
Logic:
  - Show numbers 1-20 with matching emoji count (e.g. 3 🐱🐱🐱)
  - Counting game: tap each animal as it's counted, expo-speech counts aloud
  - Simple addition: "2 🍎 + 1 🍎 = ?" with 3 answer choices
  - Number tracing (same as letter tracing above)
  - Match number to quantity (drag number to matching group of objects)

── COLORS GAME ─────────────────────────
Tech: React Native
Logic:
  - Show a colored shape and say its color (expo-speech)
  - Quiz: "Tap the RED one!" → 4 colored shapes, tap correct
  - Color mixing: "Red + Blue = ?" with animation showing mix
  - Find the color: Scene with multiple objects, "Find something YELLOW"
  - 12 colors: red, blue, yellow, green, orange, purple, pink, white, black, brown, gray, turquoise

── SHAPES GAME ─────────────────────────
Tech: react-native-svg + expo-speech
Logic:
  - Show shapes with name + sound: circle, square, triangle, rectangle, star, heart, diamond, oval
  - Shape sorter: Drag shapes into matching holes (PanResponder + collision detection)
  - Draw the shape: Child traces dotted shape outline with finger
  - Find the shape: "Tap all the CIRCLES in the scene"
  - Build a picture from shapes (house = square + triangle, etc.)

── ANIMALS GAME ────────────────────────
Tech: React Native + expo-speech + expo-av
Logic:
  - Flash card style: show animal image (Kenney Animal Pack CC0) + name
  - "Hear the sound" button plays CC0 animal sound from freesound.org
  - Quiz: "Which one says MOO?" → 4 animal images, tap the cow
  - Habitat sort: Drag animals to Ocean / Jungle / Farm / Sky
  - Animal facts read aloud via expo-speech ("Elephants are the biggest land animals!")
  - 20+ animals minimum

── PUZZLE GAME ─────────────────────────
Tech: React Native + PanResponder + react-native-svg
Logic:
  - Take a colorful image (from Kenney CC0 pack) and slice into grid pieces
  - Scramble pieces, child drags each to correct position
  - Snap-to-grid when piece is close to correct spot
  - Difficulties: 2x2 (4 pieces), 3x3 (9 pieces), 4x4 (16 pieces)
  - Ghost outline shows where each piece belongs
  - Celebrate with animation when puzzle complete
